# Preventing local development port conflicts

We need pre-determined ports for local development to prevent port conflicts as well as configuring OAuth tokens that can be used locally (client id/secret).

## Services

* cv: 8080
* cv-admin: 8081
* cv-backend: 3002
* cv-admin-backend: 3001
* postgres (cv-db): 5432
* postgres (cv-admin-db): 5433
* docling: 5001
* minio: 9000
* minio-console: 9001
